# DB-API Explorer

Consultor técnico de personajes y transformaciones basado en la API de Dragon Ball.

## Despliegue

Acceso directo a la aplicación:
https://carlosevcode.github.io/db-api/

## Características

- Filtrado dinámico por raza.
- Información técnica de personajes (Ki, Género, Raza).
- Visualización de transformaciones.
- Modal flotante para imágenes.
- Diseño minimalista en modo oscuro.

## Tecnologías

- JavaScript (Vanilla)
- CSS (Variables y Flexbox)
- HTML (Semántico)
- Fetch API

## Fuente

Datos obtenidos de: dragonball-api.com
